## Uni project

The aim of this university project was to analyze data from the tennis_atp dataset posted on github 1, which contains data on players ranked by the ATP (Association of Tennis Professionals), ATP match results with statistics, and ATP rankings since the 1970s. 

The analysis was performed using the Python 3.9 programming language, Jupyter Notebook, and libraries for data processing - pandas, numpy, scipy and visualization - matplotlib, seaborn.
